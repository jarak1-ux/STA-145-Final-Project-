# STA-145-Final-Project
My replication package for my STA-145 final project. I took the discography of the top 200 songs and  analyzed the lengths, topics, and descriptions of song. I sought to figure out if I enjoyed a certain type of song more than the others and what song I rated the highest statistically.
Instructions
To access and utilize the script, open the "STA145-JARA.K1" file, which contains the code for my project. To access the dataset, open the "UPDATED FINAL PROJ DATASET - Sheet1 (2).csv file.
Data
The data collected for this project was songs from the website that listed the top 200 songs from the 2000's. The main goal of the analysis was to determine which songs from that list do listeners enjoy the most and determining if the explicitness and certification of these songs determine the placement in the "Top 200".
